# Roblox Player Search

Search Roblox players! made by joyful_pizzapartyl.

## Shows
- Avatar profile image
- UserID
- Display name
- Description
- Join date
- Online status
- In-game status
- Currently wearing avatar items

## Removed
- Age placeholders
- Gender predict
- Account value placeholders
- Exact last seen timestamp prediction

## How to use
1. Download repo
2. Open index.html
3. Enter username
4. Click search
